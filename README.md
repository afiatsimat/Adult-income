# Adult Income Dataset - Excel Project

This project showcases data cleaning, data categorization, and formula-based analysis using the Adult Income Dataset in Microsoft Excel.

## Skills Used
- Data Cleaning
- VLOOKUP
- INDEX-MATCH
- COUNTIF & COUNTIFS
- SUMIF & SUMIFS
- AVERAGEIF & AVERAGEIFS
- Statistical Functions

## Files Included
- Raw Data
- Clean Data
- Formula Showcase
- Lookup Table

## Dataset
Adult Income Dataset (32,537 records)
